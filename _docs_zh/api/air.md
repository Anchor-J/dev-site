全国3240市县区及1700个监测站点的空气质量AQI接口，支持空气质量AQI数据，空气质量实况数据、空气质量未来7天预报。通过灵活的接口请求参数，你可以一次获取以上数据，也可以分别获取其中你需要的数据。
